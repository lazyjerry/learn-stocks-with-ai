# 第四堂：財報入門 | 第 3 節：現金流量表怎麼看

## 本節目標

- 看懂營業、投資、籌資三種現金流。
- 理解獲利與現金流不一定同步。
- 知道自由現金流為什麼重要。

重要提醒：

- 會計上賺錢，不代表現金真的進來。
- 現金流量表像公司錢包流水帳。

## 三種現金流

| 類型 | 代表什麼 |
| --- | --- |
| 營業活動現金流 | 本業營運帶進或流出的現金 |
| 投資活動現金流 | 買設備、投資、處分資產造成的現金流 |
| 籌資活動現金流 | 借錢、還錢、發股利、增資造成的現金流 |

## 營業活動現金流

營業活動現金流最重要。

它回答：

```text
公司本業有沒有真的產生現金？
```

健康狀況：

- 長期為正通常較好。
- 如果淨利成長但營業現金流長期不好，要追原因。

可能原因：

- 應收帳款增加。
- 存貨增加。
- 客戶付款變慢。
- 營運資金壓力變大。

## 投資活動現金流

投資活動現金流常常為負，因為公司買設備、擴廠或投資。

重點不是看到負數就害怕，而是問：

- 投資是否支持未來成長？
- 資本支出是否過高？
- 投資後獲利是否真的改善？

## 籌資活動現金流

籌資活動現金流反映公司如何取得或返還資金。

常見項目：

- 借款。
- 還款。
- 發放現金股利。
- 增資。

重要提醒：

- 公司發股利很好，但要看現金流是否支撐。
- 靠借錢發股利，不適合長期外推。

## 自由現金流

簡化理解：

```text
自由現金流 = 營業活動現金流 - 必要資本支出
```

它代表公司維持營運與投資後，還能自由運用的現金。

用途：

- 配息。
- 還債。
- 買回庫藏股。
- 再投資。

## 用 AI 輔助理解

```text
請根據以下現金流量表，整理營業活動、投資活動、籌資活動現金流的變化。
請判斷公司現金流是否支持目前獲利與配息，並標出需要查證的地方。
不要提供買賣建議。
```

## 記憶方法

現金流量表像錢包流水帳：

```text
損益表說你賺多少，現金流量表確認錢有沒有真的進口袋。
```

## 自我檢查

- 營業活動現金流為什麼重要？
- 投資活動現金流為負一定不好嗎？
- 籌資活動現金流能看出什麼？
- 自由現金流和配息有什麼關係？

## 本節小結

- 現金流量表能補足損益表的盲點。
- 本業現金流長期健康，比單年淨利更可靠。
- 自由現金流是配息與再投資的重要基礎。
