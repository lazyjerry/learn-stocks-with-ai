# 第四堂：財報入門 | 第 1 節：損益表怎麼看

## 本節目標

- 看懂營收、毛利、營業利益與稅後淨利。
- 理解毛利率、營益率、淨利率。
- 分辨本業獲利與一次性收益。

重要提醒：

- 損益表像公司成績單，顯示一段期間賺多少、花多少、剩多少。
- 獲利成長要確認是本業改善，不是一次性處分利益。

## 損益表的核心流程

可以先用這條線理解：

```text
營收 - 營業成本 = 毛利
毛利 - 營業費用 = 營業利益
營業利益 + 業外收支 - 稅 = 稅後淨利
```

## 營收

營收是公司賣產品或服務收到的收入。

觀察重點：

- 營收是否成長。
- 成長是價格提高、銷量增加，還是併購造成。
- 月營收與季營收是否一致。

重要提醒：

- 營收成長不等於獲利成長。
- 如果成本增加更快，賣越多也可能賺越少。

## 毛利與毛利率

毛利是營收扣掉直接成本後剩下的金額。

```text
毛利率 = 毛利 / 營收
```

毛利率反映產品競爭力與成本壓力。

毛利率上升可能代表：

- 產品組合變好。
- 售價提高。
- 成本下降。
- 規模經濟出現。

毛利率下降可能代表：

- 競爭加劇。
- 原物料上漲。
- 匯率不利。
- 低毛利產品占比提高。

## 營業利益與營益率

營業利益是毛利扣掉營業費用後的本業成果。

```text
營益率 = 營業利益 / 營收
```

營業費用通常包含：

- 銷售費用。
- 管理費用。
- 研發費用。

重要提醒：

- 營業利益比稅後淨利更能看本業。
- 稅後淨利可能受業外收入或一次性項目影響。

## 稅後淨利

稅後淨利是最後歸屬股東的獲利。

常見用途：

- 計算 EPS。
- 觀察公司整體獲利。
- 評估配息能力的一部分。

但稅後淨利要拆開看：

| 來源 | 判讀 |
| --- | --- |
| 本業獲利 | 較可持續 |
| 業外收益 | 要看是否穩定 |
| 一次性處分利益 | 不宜直接外推 |


## 記憶方法

損益表像公司成績單：

```text
營收是考了幾題，毛利是扣掉材料成本後的分數，營業利益是扣掉日常開銷後的真正本業成績。
```

## 自我檢查

- 營收成長為什麼不等於獲利成長？
- 毛利率下降可能代表什麼？
- 為什麼營業利益很重要？
- 一次性收益為什麼不能直接外推？

## 本節小結

- 損益表用來看公司一段期間的獲利表現。
- 毛利率看產品與成本，營益率看本業效率。
- 稅後淨利要分清楚本業與一次性項目。
