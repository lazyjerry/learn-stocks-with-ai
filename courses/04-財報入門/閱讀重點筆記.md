# 第四堂：財報入門 - 閱讀重點筆記

## 第 1 節：損益表怎麼看
- **核心架構**：營收 - 營業成本 = 毛利；毛利 - 營業費用 = 營業利益；營業利益 + 業外收支 - 稅 = 稅後淨利。
- **指標意義**：毛利率反映產品競爭力與成本控制，營益率反映本業經營效率，稅後淨利需區分本業持續性獲利與一次性處分利益。
- **營收 ≠ 獲利**：營收成長若成本增加更快，可能導致獲利衰退。

## 第 2 節：資產負債表怎麼看
- **基本公式**：資產 = 負債 + 股東權益，代表公司某個時間點的財務體檢表。
- **資產與負債**：需留意應收帳款與存貨是否暴增（可能代表收款不佳或庫存跌價風險）；負債不一定是壞事，但需注意短期借款與利息壓力。
- **關鍵指標**：流動比率觀察短期償債能力，負債比觀察財務槓桿（需依產業特性判斷合理範圍）。

## 第 3 節：現金流量表怎麼看
- **三大現金流**：
  - **營業活動現金流**：本業實際帶進的現金，最為重要，長期為正較為健康。
  - **投資活動現金流**：買賣設備或資產的現金流，為負不一定不好（可能在投資未來）。
  - **籌資活動現金流**：借還款、發股利、增資的現金流動。
- **自由現金流**：營業現金流扣除資本支出，代表公司真正能自由運用的現金，是評估配息能力的重要基礎。

## Codex 批註
- 整體理解正確；損益表公式是初學版。實務上稅後淨利還要分清是否歸屬母公司業主，業外收支也要拆開看是否為一次性處分利益或可持續的轉投資收益。
- 資產負債表的流動比率可搭配速動比率觀察，因為存貨不一定能快速變現。金融業的負債比天生偏高，不能直接套用一般製造業或服務業的判斷框架。
- 自由現金流有不同定義。課程使用的「營業現金流減資本支出」很適合初學者，但 DCF（現金流折現）模型常會改用 FCFF（公司自由現金流）或 FCFE（股權自由現金流），比較公司時要先確認公式一致。
