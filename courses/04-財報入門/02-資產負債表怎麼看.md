# 第四堂：財報入門 | 第 2 節：資產負債表怎麼看

## 本節目標

- 看懂資產、負債、股東權益。
- 觀察現金、存貨、應收帳款與借款。
- 理解流動比率與負債比。

重要提醒：

- 資產負債表像公司體檢表。
- 它回答的是某個時間點公司體質穩不穩。

## 基本公式

```text
資產 = 負債 + 股東權益
```

可以這樣理解：

- 資產：公司擁有或可使用的資源。
- 負債：公司欠別人的錢。
- 股東權益：扣掉負債後屬於股東的部分。

## 資產看什麼

常見重要項目：

| 項目 | 觀察重點 |
| --- | --- |
| 現金 | 公司手上安全墊 |
| 應收帳款 | 客戶欠公司的錢 |
| 存貨 | 尚未賣出的產品或材料 |
| 固定資產 | 廠房、設備等長期資產 |

重要提醒：

- 應收帳款增加太快，要確認收款品質。
- 存貨增加太快，要小心庫存跌價。

## 負債看什麼

負債不是一定不好。

合理借款可以擴張營運，但過高負債會增加壓力。

觀察重點：

- 短期借款是否過高。
- 長期借款是否穩定。
- 利息費用是否侵蝕獲利。
- 公司是否有足夠現金應付到期負債。

## 股東權益看什麼

股東權益代表公司累積下來屬於股東的淨資產。

長期觀察：

- 股東權益是否穩定增加。
- 是否頻繁增資稀釋股東。
- 保留盈餘是否累積。

## 常見指標

### 流動比率

```text
流動比率 = 流動資產 / 流動負債
```

它用來觀察短期償債能力。

### 負債比

```text
負債比 = 負債總額 / 資產總額
```

它用來觀察公司使用多少外部資金。

重要提醒：

- 負債比合理區間會因產業而異。
- 金融業、製造業、軟體業不能硬用同一標準比較。

## 用 AI 輔助理解

```text
請根據以下資產負債表資料，整理現金、應收帳款、存貨、短期借款、長期借款與股東權益的變化。
請列出可能代表體質改善的訊號，以及需要查證的風險。
```

## 記憶方法

資產負債表像體檢表：

```text
現金像血液，負債像壓力，股東權益像累積體質。
```

## 自我檢查

- 資產、負債、股東權益的關係是什麼？
- 應收帳款暴增可能有什麼風險？
- 存貨增加為什麼要搭配景氣判斷？
- 負債比高一定不好嗎？

## 本節小結

- 資產負債表看公司某一時間點的財務體質。
- 現金、應收帳款、存貨與借款是初學者優先觀察項目。
- 指標要搭配產業特性判斷。
