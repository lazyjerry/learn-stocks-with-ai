# 第八堂：風險管理與交易紀律 | 第 3 節：建立買進賣出檢查清單

> 免責聲明：本課程內容就是 AI 生成拿來學習用的內容，不構成任何投資建議、買賣建議或報酬保證。投資前請自行查證資料並評估風險。

## 本節目標

- 建立買進前檢查清單。
- 建立賣出前檢查清單。
- 用清單降低臨場情緒。

重要提醒：

- 檢查清單不是讓決策變機械，而是降低遺漏與衝動。
- 每次買賣都要能回頭檢討。

## 買進前檢查

```text
公司：
- 我知道它怎麼賺錢嗎？
- 主要客戶與產品是什麼？
- 競爭優勢是否清楚？

財報：
- 營收與獲利趨勢如何？
- 毛利率與營益率是否穩定？
- 現金流是否支持獲利？

估值：
- PER、PBR 或殖利率是否合理？
- 和歷史區間與同業相比如何？
- 悲觀情境是否仍可承受？

技術線：
- 目前趨勢如何？
- 成交量是否異常？
- 支撐壓力在哪裡？

風險：
- 原始假設失效條件是什麼？
- 停損規則是什麼？
- 單一股票部位是否過高？
```

## 賣出前檢查

```text
原始假設：
- 當初買進理由是否仍成立？
- 財報是否驗證或推翻假設？

價格：
- 估值是否已經過高？
- 是否已達樂觀情境？

風險：
- 公司或產業是否出現重大變化？
- 是否有更好的資金用途？

紀律：
- 賣出是基於規則，還是情緒？
```

> **Claude 按：** **停利紀律**和停損一樣重要，卻常被忽略。建議在買進時就設定「達到樂觀情境估值 X%」或「基本面超預期驗證 N 季後重估」的獲利了結條件。沒有預設停利條件，容易讓獲利回吐，或過早賣出錯過後續行情。把「何時賣出」與「為什麼買進」一起在筆記中寫清楚，是成熟投資紀律的核心。

## 用 AI 協助清單化

```text
請根據我的股票研究筆記，幫我產出買進前與賣出前檢查清單。
請把每個問題設計成可以回答「是 / 否 / 需要查證」。
不要提供買賣建議。
```

## 記憶方法

檢查清單像飛行前檢查：

```text
不是因為飛行員不專業，而是因為再專業的人也會漏看。
```

## 自我檢查

- 你的買進理由是否能寫成一句話？
- 你的賣出條件是否在買進前就定義？
- 你的檢查清單是否包含反方理由？
- 你是否有記錄每次決策？

## 本節小結

- 買進清單協助確認公司、財報、估值、技術線與風險。
- 賣出清單協助檢查原始假設是否失效。
- 清單能讓投資決策更可追蹤、可檢討。
