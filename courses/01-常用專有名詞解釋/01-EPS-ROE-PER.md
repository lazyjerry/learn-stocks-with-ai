# 第一堂：常用專有名詞解釋 | 第 1 節：EPS、ROE、PER

## 本節目標

讀完這一節，你要能做到三件事：

- 看懂財經文章常出現的 EPS、ROE、PER。
- 知道這三個指標各自回答什麼問題。
- 用 AI 協助解讀指標時，能問出比較不容易被誤導的問題。

重要提醒：

- 本課是教育內容，不是投資建議。
- 單一指標不能決定一檔股票能不能買。
- 數字要搭配產業、公司成長性、景氣循環與股價位置一起看。

## EPS：每一股賺多少錢

EPS 是 Earnings Per Share（每股盈餘）。

公式：

```text
EPS = 稅後淨利 / 流通在外股數
```

它回答的問題是：

```text
公司把獲利平均分到每一股後，每一股大約賺多少錢？
```

例子：

```text
A 公司今年稅後淨利：10 億元
A 公司流通在外股數：1 億股

EPS = 10 億元 / 1 億股 = 10 元
```

這代表 A 公司今年每一股大約賺 10 元。

### 怎麼解讀 EPS

EPS 上升，通常代表公司獲利能力變好；EPS 下降，通常代表獲利轉弱。

但要先問：

- EPS 成長是來自本業，還是一次性收益？
- EPS 成長是營收真的變好，還是成本暫時下降？
- EPS 是否連續多年穩定成長？
- 公司有沒有大量增資，導致股數變多、稀釋 EPS？

錯誤解讀：

```text
EPS 高，所以股票一定值得買。
```

比較好的解讀：

```text
EPS 高代表公司賺錢能力不錯，但還要看股價、產業成長、獲利穩定度與估值。
```

## ROE：股東的錢被用得好不好

ROE 是 Return on Equity（股東權益報酬率）。

公式：

```text
ROE = 稅後淨利 / 股東權益
```

它回答的問題是：

```text
公司拿股東投入與累積下來的資本，創造了多少報酬？
```

例子：

```text
B 公司股東權益：100 億元
B 公司稅後淨利：15 億元

ROE = 15 億元 / 100 億元 = 15%
```

這代表 B 公司用股東資金創造出 15% 的報酬。

### 怎麼解讀 ROE

ROE 越高，通常代表公司使用股東資金的效率越好。

但高 ROE 不一定全是好事，可能有三種來源：

- 本業獲利能力強。
- 資產使用效率好。
- 負債比例高，用槓桿放大報酬。

重要提醒：

- ROE 高，要同步看負債比。
- 如果 ROE 高是因為負債很重，景氣反轉時壓力會變大。
- ROE 要看長期穩定度，不要只看單一年。

## PER：市場願意用幾倍獲利買這家公司

PER 是 Price-to-Earnings Ratio（本益比），也常寫成 P/E Ratio。

公式：

```text
PER = 股價 / EPS
```

它回答的問題是：

```text
市場現在願意用幾倍的每股獲利來買這家公司？
```

例子：

```text
C 公司股價：100 元
C 公司 EPS：5 元

PER = 100 元 / 5 元 = 20 倍
```

這代表市場願意用 20 倍獲利買 C 公司。

### 怎麼解讀 PER

PER 低，不一定代表便宜；PER 高，也不一定代表昂貴。

PER 低可能代表：

- 股價被低估。
- 市場認為公司未來成長會變差。
- 公司處在景氣循環高峰，現在獲利很高但不可持續。

PER 高可能代表：

- 市場看好未來成長。
- 股價過熱。
- 短期 EPS 偏低，讓 PER 看起來變高。

重要提醒：

- PER 不能跨產業硬比。
- 成熟產業與高速成長產業的合理 PER 可能差很多。
- 景氣循環股在高獲利時 PER 可能看起來很低，但不一定便宜。

## 三個指標放在一起看

| 指標 | 問題 | 觀察重點 |
| --- | --- | --- |
| EPS | 每一股賺多少？ | 獲利能力與成長趨勢 |
| ROE | 股東資金用得好不好？ | 經營效率與財務槓桿 |
| PER | 股價相對獲利貴不貴？ | 市場預期與估值水位 |

一個簡單的判讀流程：

```text
EPS 看獲利
ROE 看效率
PER 看價格
```

例子：

```text
公司甲：
EPS 連續 5 年成長
ROE 長期維持 15% 以上
PER 目前低於過去 5 年平均
```

這種情況值得進一步研究，但還不能直接下買進結論。

還要查：

- 成長原因是否可持續。
- 負債是否合理。
- 產業是否正在轉弱。
- 股價技術線是否已經過熱。
- 是否有重大一次性收益扭曲 EPS。

## 用 AI 輔助理解

可以這樣問 AI：

```text
請用初學者能理解的方式，解釋 EPS、ROE、PER 的差異。
請用一家公司作為假設例子，說明這三個指標分別代表什麼。
最後請列出使用這三個指標時最常見的 5 個誤解。
```

分析個股時，可以這樣問：

```text
我正在研究一家公司。
請根據我提供的 EPS、ROE、PER 數據，幫我分成「事實」、「可能推論」、「需要進一步查證」三欄。
不要直接給買賣建議。

資料如下：
公司：
年度 EPS：
年度 ROE：
目前 PER：
產業：
```

重要提醒：

- AI 如果沒有資料來源，答案只能當草稿。
- 請要求 AI 標出「不知道」與「需要查證」的地方。
- 投資分析要保留資料日期，因為 EPS、ROE、PER 都會隨財報與股價變動。

## 記憶方法

EPS：每一股的考試分數。

```text
公司越會賺錢，每一股拿到的分數通常越高。
```

ROE：老闆拿股東的錢去做生意，轉出多少報酬。

```text
同樣拿 100 元，有人賺 5 元，有人賺 20 元，效率差很多。
```

PER：市場願意用幾年獲利買下這家公司。

```text
PER 20 倍，可以粗略想成市場願意用 20 年目前獲利的價格買它。
```

重要提醒：

- PER 的「幾年回本」只是記憶方法，不是精準估值。
- 公司獲利會變，未來 EPS 不一定等於現在 EPS。

## 自我檢查

回答下面問題，確認自己是否理解：

- EPS 高，為什麼不代表股票一定便宜？
- ROE 高時，為什麼要同步看負債？
- PER 低時，可能有哪些風險？
- 如果一家公司 EPS 成長、ROE 穩定、PER 偏低，下一步還要查什麼？

## 本節小結

- EPS 看公司每一股賺多少錢。
- ROE 看公司使用股東資金的效率。
- PER 看市場願意用幾倍獲利買這家公司。
- 這三個指標要一起看，也要搭配產業、財報品質與股價位置。
