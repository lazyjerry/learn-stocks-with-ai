# 第一堂：常用專有名詞解釋 | 第 2 節：加權指數、大盤、成交量

## 本節目標

讀完這一節，你要能做到三件事：

- 理解加權指數與大盤在講什麼。
- 知道成交量為什麼要跟價格一起看。
- 看財經新聞時，不再把「大盤上漲」誤解成「所有股票都上漲」。

重要提醒：

- 指數代表市場概況，不代表每一檔股票的狀況。
- 成交量代表交易熱度，不直接等於股價一定會上漲或下跌。
- 技術線圖會在第七堂深入，本節只先建立基本詞彙。

## 加權指數：台股市場的主要溫度計

台灣加權股價指數常被稱為加權指數，也有人直接說台股大盤指數。

它用來反映台灣上市股票整體表現，但不是每家公司影響力都一樣。

「加權」的意思是：

```text
市值越大的公司，對指數影響越大。
```

市值可以先粗略理解為：

```text
市值 = 股價 x 發行股數
```

所以大型權值股漲跌，會比小型股更影響加權指數。

### 例子

假設一個簡化市場只有兩家公司：

| 公司 | 市值 | 今天漲跌 | 對指數影響 |
| --- | ---: | ---: | --- |
| A 公司 | 9,000 億元 | 上漲 1% | 很大 |
| B 公司 | 100 億元 | 下跌 5% | 較小 |

就算 B 公司跌很多，只要 A 公司市值夠大，整體指數仍可能上漲。

重要提醒：

- 加權指數上漲，不代表你手上的股票一定會漲。
- 如果指數靠少數大型股撐住，很多中小型股仍可能下跌。

## 大盤：市場整體氣氛的口語說法

投資人常說「今天大盤很強」、「大盤轉弱」、「大盤跌破月線」。

這裡的大盤通常指：

```text
整體股票市場的走勢與氣氛。
```

大盤可以用不同方式觀察：

- 加權指數漲跌。
- 上漲家數與下跌家數。
- 成交金額是否放大。
- 類股輪動狀況。
- 外資、投信、自營商買賣超。

初學者先掌握一個原則：

```text
大盤是背景，不是個股答案。
```

個股分析仍要回到公司本身：

- 產業位置。
- 財報表現。
- 估值水位。
- 技術線位置。
- 籌碼與市場情緒。

## 成交量：市場參與的音量

成交量代表某一段時間內股票被交易的數量。

在個股中，成交量常用「張」表示。

```text
1 張 = 1,000 股
```

如果某檔股票今天成交 10,000 張，代表今天有 10,000 張股票完成買賣。

成交量回答的問題是：

```text
今天有多少人願意在這個價格附近買賣？
```

### 成交量不是只有大或小

成交量要搭配價格位置判斷。

| 狀況 | 初步解讀 |
| --- | --- |
| 價漲量增 | 上漲有資金參與，趨勢可能較有力 |
| 價漲量縮 | 上漲力道可能不足，也可能只是賣壓少 |
| 價跌量增 | 賣壓變大，或有人趁下跌大量換手 |
| 價跌量縮 | 市場興趣降低，或賣壓暫時減弱 |

重要提醒：

- 上表只是初步觀察，不是固定規則。
- 同樣是爆量，出現在低檔、高檔、突破、跌破，意義都不同。

## 指數、個股、成交量一起看

初學者可以用三層架構理解：

```text
加權指數：市場整體方向
大盤氣氛：多數股票與資金狀態
成交量：買賣參與程度
```

例子：

```text
情境 A：
加權指數上漲 1%
但下跌家數多於上漲家數
成交量集中在少數大型權值股
```

可能解讀：

```text
指數看起來強，但市場內部不一定全面健康。
```

例子：

```text
情境 B：
加權指數小漲
多數類股輪流上漲
成交量溫和放大
```

可能解讀：

```text
市場參與度較廣，氣氛比單靠少數權值股拉抬更穩。
```

## 用 AI 輔助理解

可以這樣問 AI：

```text
請用初學者能理解的方式，解釋加權指數、大盤、成交量的差異。
請用台股作為情境，但不要提供個股買賣建議。
最後請列出常見誤解與正確理解。
```

觀察市場時，可以這樣問：

```text
我提供今天的大盤資料，請幫我整理成「事實」、「可能代表的市場氣氛」、「不能直接推論的事情」三欄。

資料：
加權指數漲跌：
成交金額：
上漲家數：
下跌家數：
強勢類股：
弱勢類股：
```

重要提醒：

- AI 沒有即時資料時，不要讓它憑空判斷今天大盤。
- 市場資料會即時變動，使用 AI 前要確認資料日期與時間。
- 不要只問「明天會不會漲」，這種問題通常會得到看似完整但不可驗證的回答。

## 記憶方法

加權指數：班級總平均，但大同學權重更高。

```text
大型權值股像班上影響總平均最多的學生。
```

大盤：整個市場的天氣。

```text
天氣晴朗不代表每個人都不會淋雨；大盤好也不代表每檔股票都好。
```

成交量：市場音量。

```text
音量變大，代表更多人在這個價格附近表態。
```

重要提醒：

- 成交量放大代表分歧或共識變強，但要看價格方向與位置。
- 指數很強時，更要檢查自己的股票是不是跟得上。

## 自我檢查

回答下面問題，確認自己是否理解：

- 為什麼加權指數上漲，不代表所有股票都上漲？
- 市值大的公司為什麼對加權指數影響更大？
- 成交量放大一定是好事嗎？
- 看到「大盤很強」時，你還需要檢查哪些資訊？

## 本節小結

- 加權指數是台股市場的重要參考，但大型權值股影響較大。
- 大盤是市場整體氣氛，不是個股買賣答案。
- 成交量代表市場參與程度，要和價格方向、價格位置一起看。
- 初學者先學會區分「市場背景」與「個股狀況」。
