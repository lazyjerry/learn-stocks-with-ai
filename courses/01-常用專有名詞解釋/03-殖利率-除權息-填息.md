# 第一堂：常用專有名詞解釋 | 第 3 節：殖利率、除權息、填息

## 本節目標

讀完這一節，你要能做到三件事：

- 理解殖利率、除權息、填息的基本意思。
- 知道高殖利率為什麼不一定等於低風險。
- 看懂存股討論中常見的配息與填息說法。

重要提醒：

- 股利不是免費多出來的錢。
- 高殖利率要搭配獲利穩定度、現金流與股價風險一起看。
- 除權息與填息是價格調整與市場交易結果，不是保證獲利。

## 殖利率：股利相對股價的比例

殖利率可以先理解成：

```text
用目前股價買進股票，現金股利大約占股價的比例。
```

常見公式：

```text
現金殖利率 = 每股現金股利 / 股價
```

例子：

```text
A 公司股價：50 元
A 公司今年每股發放現金股利：3 元

現金殖利率 = 3 / 50 = 6%
```

這代表如果你用 50 元買進，並拿到 3 元現金股利，表面殖利率是 6%。

### 殖利率怎麼看

殖利率高，常被解讀成股利報酬較高。

但要問：

- 股利是否穩定？
- 股利是由本業獲利支撐，還是靠處分資產或借錢？
- 公司自由現金流是否足夠？
- 股價是否因為基本面變差而下跌，才讓殖利率看起來變高？

錯誤解讀：

```text
殖利率 8%，所以比定存好，一定適合存股。
```

比較好的解讀：

```text
殖利率 8% 值得研究，但要確認股利是否可持續，以及股價下跌風險是否過高。
```

## 除權息：股利發放前後的價格調整

公司發放股利時，常見兩種：

- 現金股利：公司發現金給股東。
- 股票股利：公司發股票給股東。

除息通常指配發現金股利後，股價會扣掉相對金額。

例子：

```text
B 公司除息前股價：100 元
B 公司配發現金股利：5 元

除息參考價大約是 95 元
```

這不是市場突然看壞公司，而是股利從公司價值中分出一部分給股東。

### 股利不是免費午餐

除息前：

```text
你持有一股，股價 100 元。
```

除息後：

```text
你持有一股，參考股價約 95 元。
你會收到 5 元現金股利。
```

表面上資產沒有憑空增加，只是形式從部分股價變成現金。

重要提醒：

- 實際股價會由市場交易決定，不一定剛好等於除息參考價。
- 現金股利入帳時間與除息日不同。
- 股利所得可能涉及稅務，實際報酬要考慮稅負與交易成本。

## 填息：股價補回除息缺口

填息是指股票除息後，股價漲回除息前的價格。

例子：

```text
C 公司除息前股價：100 元
C 公司配發現金股利：5 元
除息參考價：約 95 元

如果之後股價漲回 100 元，就稱為填息。
```

填息代表市場願意用買盤把除息造成的價格缺口補回來。

### 為什麼填息重要

對存股族來說，只看拿到多少股利不夠。

還要看：

- 除息後股價是否能穩定回升。
- 公司獲利是否足以支撐未來股利。
- 市場是否認同公司的長期價值。

如果拿到股利，但股價長期填不了息，總報酬可能不如想像。

## 殖利率、除息、填息一起看

可以用一個流程理解：

```text
殖利率：我用這個價格買，股利報酬看起來多少？
除息：公司把股利分出來，股價先做相對調整。
填息：市場之後是否願意把股價買回除息前水準？
```

例子：

```text
公司甲：
股價 80 元
預計配息 4 元
殖利率 5%
過去 5 年有 4 年完成填息
營業現金流穩定
```

這種情況可以進一步研究，但仍要確認：

- 今年獲利是否延續。
- 配息率是否過高。
- 產業景氣是否轉弱。
- 股價是否已經提前反映利多。

## 常見誤解

### 誤解一：高殖利率就是安全

高殖利率可能來自高股利，也可能來自股價大跌。

如果股價從 100 元跌到 50 元，原本 5 元股利的殖利率會從 5% 變成 10%。

但股價大跌可能代表市場正在擔心公司未來。

### 誤解二：除息後一定會填息

填息需要市場買盤支持。

如果公司基本面轉弱、景氣下滑或市場風險升高，股價可能長期填不了息。

### 誤解三：領股利就是賺到

領到股利只是總報酬的一部分。

總報酬要看：

```text
股價變化 + 股利收入 - 稅負 - 交易成本
```

## 用 AI 輔助理解

可以這樣問 AI：

```text
請用初學者能理解的方式，解釋殖利率、除息、填息。
請用一個假設股價與股利的例子，說明除息前後投資人資產如何變化。
最後列出高殖利率股票最常見的風險。
```

分析配息資料時，可以這樣問：

```text
我正在研究一家公司是否適合長期追蹤。
請根據我提供的資料，幫我整理「股利穩定性」、「填息紀錄」、「需要查證的風險」。
不要直接給買賣建議。

資料：
公司：
近 5 年 EPS：
近 5 年現金股利：
近 5 年除息後是否填息：
目前股價：
目前殖利率：
營業現金流：
```

重要提醒：

- AI 對除權息資料可能會過期，請用證交所、櫃買中心、公司公告或券商資料確認。
- 配息紀錄要看多年，不要只看一年。
- 需要區分「殖利率看起來高」與「股利真的可持續」。

## 記憶方法

殖利率：存股的租金率。

```text
你買進一間會配息的公司，股利像租金，但房價本身也會波動。
```

除息：公司把一部分價值發給你，股價先扣掉那一塊。

```text
口袋從左邊搬一部分到右邊，不是憑空多出來。
```

填息：股價把除息留下的洞補回來。

```text
除息像地上出現一個小坑，填息就是市場把坑補平。
```

重要提醒：

- 「租金率」只是記憶方法，股票不是定存，也不是房地產。
- 股利穩定與股價穩定是兩件事。

## 自我檢查

回答下面問題，確認自己是否理解：

- 為什麼高殖利率不一定代表低風險？
- 除息後股價為什麼會調整？
- 填息代表什麼？
- 如果一家公司殖利率很高，但連續多年沒有填息，你會追查哪些問題？

## 本節小結

- 殖利率是股利相對股價的比例。
- 除權息是公司配發股利後，股價做相對調整。
- 填息是除息後股價漲回除息前水準。
- 存股不能只看殖利率，要看獲利、現金流、配息穩定度與填息能力。
