# 第六堂：估值與合理價 | 第 2 節：股利折現與現金流折現概念

## 本節目標

- 理解股利折現與 DCF 的基本想法。
- 知道折現率與成長率會大幅影響結果。
- 把折現模型當成假設工具，而不是精準答案。

重要提醒：

- 折現模型看起來精準，但假設偏掉，結論會差很多。
- 初學者先理解概念，不必急著追求複雜公式。

## 股利折現

股利折現適合穩定配息公司。

核心想法：

```text
股票價值來自未來可收到的股利，把未來股利折回今天。
```

適合公司：

- 配息穩定。
- 獲利成熟。
- 成長率不劇烈。
- 現金流穩定。

不適合：

- 不配息公司。
- 高成長但現金流不穩定公司。
- 景氣循環劇烈公司。

## DCF：現金流折現

DCF 是 Discounted Cash Flow（現金流折現）。

核心想法：

```text
公司價值來自未來能創造的自由現金流。
```

需要假設：

- 未來營收成長。
- 毛利率與費用率。
- 資本支出。
- 自由現金流。
- 折現率。
- 長期成長率。

## 折現率是什麼

折現率可以想成你要求的報酬率。

風險越高，折現率通常越高。

```text
風險高的公司，未來現金流要打更大的折扣。
```

## 成長率為什麼危險

成長率只要差一點，估值結果就可能差很多。

常見錯誤：

- 把短期高成長外推到長期。
- 忽略產業景氣循環。
- 忽略競爭造成毛利率下降。
- 忽略資本支出需求。


## 本節小結

- 模型結果要用情境比較，不要當成唯一答案。
