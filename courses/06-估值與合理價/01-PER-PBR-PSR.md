# 第六堂：估值與合理價 | 第 1 節：PER、PBR、PSR

> 免責聲明：本課程內容就是 AI 生成拿來學習用的內容，不構成任何投資建議、買賣建議或報酬保證。投資前請自行查證資料並評估風險。

## 本節目標

- 用 PER、PBR、PSR 初步判斷價格水位。
- 知道不同估值指標適合不同公司。
- 避免只用單一倍數判斷便宜或昂貴。

重要提醒：

- 估值指標要搭配產業、景氣循環與公司成長性。
- 低估值不一定便宜，高估值不一定昂貴。

## PER：用獲利衡量價格

```text
PER = 股價 / EPS
```

PER 適合獲利穩定的公司。

觀察方式：

- 和自己歷史 PER 比。
- 和同業 PER 比。
- 搭配獲利成長率看。

常見誤解：

```text
PER 低就便宜。
```

可能真相：

- 市場預期獲利下滑。
- 公司位在景氣高峰。
- 產業風險升高。

> **Claude 按：** 實務上要區分**Trailing P/E**（歷史本益比，用過去 12 個月實際 EPS）與**Forward P/E**（預估本益比，用市場預估未來 12 個月 EPS）。法人估值更重 Forward P/E；媒體引用多為 Trailing P/E。另，**PEG**（PER ÷ 盈餘成長率）常用於評估成長股，PEG 接近 1 有時被視為「成長合理定價」的粗估基準；PEG < 1 可能代表成長被低估，PEG > 2 則需特別謹慎。

## PBR：用帳面淨值衡量價格

```text
PBR = 股價 / 每股淨值
```

PBR 適合資產價值重要的產業，例如金融、資產型公司。

觀察方式：

- PBR 是否低於歷史平均。
- ROE 是否足以支持較高 PBR。
- 資產品質是否可靠。

重要提醒：

- PBR 低不一定安全，資產可能需要打折。
- 高 ROE 公司常能享有較高 PBR。

> **Claude 按：** 評估 PBR 時要留意**無形資產與商譽**：帳面淨值包含商譽、智慧財產等無形資產，公司體質惡化時可能大額減損，使 PBR 看起來偏低但實際不安全。部分分析師改用「**有形帳面淨值**」（Tangible Book Value，扣除無形資產後）計算 PBR，對評估金融業尤其重要。

## PSR：用營收衡量價格

```text
PSR = 市值 / 年營收
```

PSR 常用於獲利不穩定、但營收仍有參考價值的公司。

適合情境：

- 新興成長公司。
- 虧損但營收快速成長的公司。
- 毛利率與商業模式需要進一步觀察的公司。

重要提醒：

- 營收不等於獲利。
- PSR 要搭配毛利率與未來獲利能力。

> **Claude 按：** 半導體、雲端等高毛利行業，分析師常改用 **EV/Sales**（Enterprise Value / Sales）取代 P/S，因為 EV = 市值 + 有息負債 - 現金及約當現金，比單純市值更能反映公司整體定價，消除不同資本結構的影響。同理，EV/EBITDA 比 PER 更常用於跨國或跨資本結構的比較。

## 三個指標怎麼選

| 指標 | 看什麼 | 較適合 |
| --- | --- | --- |
| PER | 獲利 | 獲利穩定公司 |
| PBR | 淨值 | 金融、資產型公司 |
| PSR | 營收 | 獲利不穩定成長股 |

記憶：

```text
PER 看賺錢
PBR 看家底
PSR 看生意規模
```

## 本節小結

- PER、PBR、PSR 是常見估值起點。
- 不同產業適合不同指標。
- 估值不是找最低倍數，而是判斷價格是否合理反映風險與成長。
