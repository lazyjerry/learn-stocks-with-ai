# 第六堂：估值與合理價 | 第 3 節：建立悲觀、中性、樂觀情境

## 本節目標

- 用三種情境避免只看單一目標價。
- 學會寫清楚營收、毛利率、PER 或現金流假設。
- 理解合理價是推導結果，不是預言。

重要提醒：

- 目標價不是預言，是假設推導結果。
- 假設改變，合理價也要更新。

## 為什麼要做三種情境

只做一個目標價很危險，因為它會讓人誤以為未來只有一種可能。

三種情境：

- 悲觀：成長不如預期，估值下修。
- 中性：公司照合理預期發展。
- 樂觀：成長超預期，估值上修。

## 情境要寫什麼

每個情境至少寫：

- 營收成長假設。
- 毛利率或營益率假設。
- EPS 或自由現金流假設。
- 合理 PER、PBR 或折現率。
- 主要風險。

範例：

```text
悲觀情境：
營收年減 5%
毛利率下降 2 個百分點
EPS 為 4 元
合理 PER 12 倍
推估價格 48 元
```

## 不要只看樂觀情境

很多投資錯誤來自只相信最好的劇本。

檢查問題：

- 如果營收沒有成長，還能接受嗎？
- 如果毛利率下降，合理價會掉多少？
- 如果市場只給較低 PER，安全邊際夠嗎？

## 用 AI 輔助建立情境

```text
請根據我提供的財報與估值資料，建立悲觀、中性、樂觀三種情境。
每個情境請列出營收、毛利率、EPS、合理 PER、推估價格與主要假設。
請標示哪些假設最敏感。
不要提供買賣建議。
```

## 記憶方法

情境分析像看天氣預報：

```text
不是只準備晴天，也要知道下雨和颱風時會怎樣。
```

## 自我檢查

- 為什麼只看單一目標價很危險？
- 情境分析至少要列出哪些假設？
- 哪些假設最容易影響合理價？
- 悲觀情境的用途是什麼？

## 本節小結

- 三種情境能讓估值更有彈性。
- 合理價要寫清楚假設。
- 投資前要先知道自己承受得起哪種壞情境。
