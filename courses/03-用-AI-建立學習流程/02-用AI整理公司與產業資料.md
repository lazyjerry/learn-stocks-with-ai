# 第三堂：用 AI 建立學習流程 | 第 2 節：用 AI 整理公司與產業資料

## 本節目標

- 用 AI 整理公司產品、客戶、競爭對手與產業循環。
- 把新聞拆成事實、公司說法、媒體推論與市場反應。
- 建立產業關鍵字清單，方便後續追蹤。

重要提醒：

- AI 產出的公司資料要回到公開資訊觀測站、年報、法說會資料確認。
- 新聞不是全部事實，標題常常有情緒與推論。

## 公司資料要看什麼

研究一家公司，先整理這些欄位：

| 欄位 | 要回答的問題 |
| --- | --- |
| 主要產品 | 公司賣什麼？ |
| 收入來源 | 錢從哪裡來？ |
| 主要客戶 | 客戶集中還是分散？ |
| 競爭對手 | 跟誰競爭？ |
| 成本結構 | 成本受哪些因素影響？ |
| 產業位置 | 是龍頭、追隨者，還是利基廠商？ |

AI 可以協助整理，但你要提供資料來源。

範例 Prompt：

```text
請根據以下公司年報段落，整理這家公司的主要產品、收入來源、客戶類型、競爭對手與主要風險。
請只使用我提供的資料，不要自行補充。
```

## 產業資料要看什麼

產業資料至少分成五類：

- 需求來源：誰需要這個產品？
- 供給狀況：競爭者多不多？
- 景氣循環：需求是否容易大起大落？
- 成本變化：原物料、匯率、利率是否影響大？
- 政策與技術：政策補助或技術變化是否重要？

範例 Prompt：

```text
請幫我建立這個產業的研究框架：
產業名稱：
請輸出：
1. 需求來源
2. 供給狀況
3. 景氣循環
4. 成本變數
5. 政策與技術變化
6. 需要追蹤的關鍵字
```

## 新聞要拆開看

新聞整理不要只看標題。

建議拆成四欄：

| 類型 | 說明 |
| --- | --- |
| 事實 | 已公告或可驗證的內容 |
| 公司說法 | 公司管理層或新聞稿中的描述 |
| 媒體推論 | 記者或市場人士的解讀 |
| 市場反應 | 股價、成交量、法人買賣變化 |

範例 Prompt：

```text
請把以下新聞拆成四欄：事實、公司說法、媒體推論、市場反應。
請標示哪些內容需要進一步查證。
新聞內容如下：
```

## 建立產業關鍵字清單

關鍵字清單能幫你後續追蹤。

範例：

```text
產業：半導體設備
關鍵字：資本支出、先進製程、晶圓廠擴產、設備交期、庫存修正、出口管制
```

請 AI 協助建立關鍵字：

```text
請針對「產業名稱」建立 20 個追蹤關鍵字。
請分成需求、供給、成本、政策、技術、風險六類。
```

## 重要提醒

- 客戶集中度高的公司，要注意單一客戶砍單風險。
- 產業景氣循環強的公司，要避免只看景氣高峰財報。
- 公司說法要和後續財報驗證，不能只聽故事。

## 記憶方法

公司研究像認識一間店：

```text
賣什麼、賣給誰、跟誰競爭、成本怎麼來、客人會不會回來。
```

產業研究像看商圈：

```text
同一間店在熱門商圈與冷清商圈，未來差很多。
```

## 自我檢查

- 公司資料最少要整理哪些欄位？
- 新聞為什麼不能只看標題？
- 產業關鍵字有什麼用途？
- AI 整理出的資料要去哪裡查證？

## 本節小結

- AI 適合幫你把公司與產業資料結構化。
- 新聞要拆成事實、說法、推論與反應。
- 產業關鍵字能提高後續追蹤效率。
