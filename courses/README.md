# 課程索引

這份索引提供正式課程的閱讀順序與各節連結。建議照堂數閱讀；如果已有基礎，也可以直接跳到需要的主題。

重要提醒：

- 本系列是投資教育內容，不是投資建議。
- 每篇文章都要搭配實際資料來源與日期使用。
- AI 輸出應視為研究草稿，不能取代資料查證與個人判斷。

## 建議學習順序

```text
詞彙理解
→ 股票市場基本觀念
→ AI 輔助研究流程
→ 財報閱讀
→ 財務指標
→ 估值
→ 技術線圖
→ 風險管理
→ 完整案例練習
```

## 建議使用方式

- 每讀完一節，先回答文末自我檢查。
- 讀到第三堂後，開始建立自己的股票筆記模板。
- 讀到第四堂後，挑一家公司練習看三大財報。
- 讀到第六堂後，練習建立悲觀、中性、樂觀三種情境。
- 讀到第九堂後，完成一份完整投資研究筆記。
