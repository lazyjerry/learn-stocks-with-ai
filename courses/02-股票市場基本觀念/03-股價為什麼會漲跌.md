# 第二堂：股票市場基本觀念 | 第 3 節：股價為什麼會漲跌

## 本節目標

讀完這一節，你要能做到三件事：

- 理解股價短期與長期的主要驅動因素。
- 分清楚消息、資金、情緒與基本面的差異。
- 用 AI 分析漲跌原因時，避免把推論當成事實。

重要提醒：

- 股價漲跌常常有多個原因，不一定能用單一事件解釋。
- 市場常先反應預期，再等財報或新聞確認。
- 「知道為什麼漲」不等於「知道接下來會繼續漲」。

## 股價由買賣決定

股價最直接的形成方式是市場買賣。

當願意買的人更積極，願意用更高價格買，股價就容易上漲。

當願意賣的人更積極，願意用更低價格賣，股價就容易下跌。

可以先這樣理解：

```text
股價是買方與賣方每天重新協商出來的價格。
```

這個價格會受到很多因素影響：

- 公司獲利。
- 產業景氣。
- 市場資金。
- 利率與匯率。
- 新聞消息。
- 投資人情緒。
- 技術線位置。
- 籌碼變化。

## 短期：資金、消息與情緒

短期股價常被三件事影響：

```text
資金 + 消息 + 情緒
```

### 資金

如果市場資金充足，投資人願意承擔風險，股價比較容易被推高。

可能觀察：

- 大盤成交金額放大。
- 熱門類股輪動。
- 外資或投信買超。
- 題材股成交量增加。

重要提醒：

- 資金行情可能推升股價，但不一定代表公司獲利同步改善。

### 消息

消息可能讓市場快速改變預期。

常見消息：

- 新產品。
- 接到大訂單。
- 法說會展望。
- 財報優於預期。
- 政策補助。
- 產業缺貨或漲價。

消息要分清楚：

| 類型 | 解讀 |
| --- | --- |
| 已公告事實 | 可回到公開資料確認 |
| 公司說法 | 需要觀察後續是否兌現 |
| 媒體推論 | 只能當線索 |
| 市場傳言 | 風險最高，不能直接當依據 |

### 情緒

市場情緒會讓股價短期偏離基本面。

常見情緒：

- 貪婪：大家怕錯過上漲。
- 恐慌：大家急著賣出避險。
- 從眾：看到別人買就跟著買。
- 過度自信：只看自己想看的資料。

重要提醒：

- 短期股價常反應情緒，但情緒很難精準預測。

## 長期：獲利、成長與現金流

長期來看，股價通常會回到公司能否持續創造價值。

更接近長期核心的是：

```text
獲利能力 + 成長性 + 現金流 + 風險
```

### 獲利能力

公司能不能穩定賺錢，是長期股價的重要基礎。

可以觀察：

- EPS 是否長期成長。
- 毛利率是否穩定。
- 營業利益是否來自本業。
- ROE 是否穩定。

### 成長性

市場買的是未來。

如果市場預期公司未來獲利會成長，股價可能先上漲。

但如果成長不如預期，股價也可能快速修正。

### 現金流

會計上賺錢，不代表現金真的進來。

長期投資要注意：

- 營業活動現金流是否穩定。
- 公司是否長期自由現金流為正。
- 配息是否有現金流支撐。

重要提醒：

- 長期股價不是每天都跟基本面同步，但基本面會影響市場願意給的價格。

## 預期比已知事實更早反應

股價常常不是等好消息公布才上漲，而是先反應市場預期。

例子：

```text
市場預期公司下季獲利會大幅成長
股價先漲一段
財報公布後真的成長
但因為沒有超過預期，股價反而下跌
```

這就是常聽到的：

```text
利多出盡
```

意思不是消息不好，而是好消息已經提前反映在股價裡。

重要提醒：

- 好消息不一定讓股價上漲。
- 壞消息不一定讓股價下跌。
- 關鍵在於「是否超過或低於市場原本預期」。

## 好公司、好價格、好時機

分析股價漲跌時，可以拆成三個問題：

| 問題 | 觀察重點 |
| --- | --- |
| 公司好不好 | 財報、產業、競爭力 |
| 價格合不合理 | PER、PBR、殖利率、歷史區間 |
| 時機是否合適 | 大盤、技術線、成交量、風險事件 |

初學者常犯的錯：

```text
只因為公司很好，就忽略股價是否太貴。
```

比較好的想法：

```text
好公司要搭配合理價格與可承受的風險。
```

## 用 AI 輔助理解

可以這樣問 AI：

```text
請用初學者能理解的方式，解釋股價短期與長期為什麼會漲跌。
請把原因分成「資金」、「消息」、「情緒」、「基本面」四類。
最後列出每一類不能直接推論的事情。
```

分析個股漲跌時，可以這樣問：

```text
我提供一檔股票近期資料。
請幫我整理可能影響股價的因素，分成「已確認事實」、「可能推論」、「需要查證」。
不要預測明天漲跌，也不要給買賣建議。

股票：
近期漲跌幅：
新聞事件：
最近財報：
成交量變化：
大盤狀況：
產業消息：
```

重要提醒：

- AI 很容易把相關事件串成看似合理的故事。
- 沒有證據時，要讓 AI 標示為「可能推論」。
- 漲跌原因常常只能事後部分解釋，不能保證下一次也一樣。

## 記憶方法

股價：現在成績加未來想像。

```text
財報是現在成績，市場預期是未來想像。
股價常常同時反映兩者。
```

短期：市場心情。

```text
短期股價像市場今天的心情，會受消息與資金影響。
```

長期：公司體質。

```text
時間拉長後，公司能不能賺錢、能不能成長，會越來越重要。
```

## 本節小結

- 股價由市場買賣形成，背後受到資金、消息、情緒與基本面影響。
- 短期股價容易受市場情緒與資金推動。
- 長期股價更離不開獲利、成長與現金流。
- 市場常先反應預期，投資人要避免把事後故事當成必然規律。
