# 第五堂：財務指標與公司體質 | 第 3 節：自由現金流與股東權益

## 本節目標

- 理解自由現金流代表什麼。
- 用股東權益觀察公司長期累積價值。
- 判斷公司是健康投資，還是現金壓力。

重要提醒：

- 長期自由現金流為負，要確認是成長投資還是經營壓力。
- 股東權益增加，通常代表公司長期累積價值，但仍要看品質。

## 自由現金流是什麼

簡化公式：

```text
自由現金流 = 營業活動現金流 - 資本支出
```

它代表公司維持營運與必要投資後，還能自由運用的現金。

用途包含：

- 發股利。
- 還債。
- 投資新事業。
- 買回庫藏股。

## 自由現金流怎麼看

健康訊號：

- 長期為正。
- 與獲利方向一致。
- 足以支撐配息。
- 不需要一直靠借款補現金。

需要追查：

- 獲利成長但自由現金流長期為負。
- 配息高於自由現金流。
- 資本支出大增但未來報酬不明。

## 股東權益是什麼

股東權益代表公司扣掉負債後屬於股東的淨資產。

```text
股東權益 = 資產 - 負債
```

長期股東權益增加，可能代表公司持續累積價值。

但也要確認：

- 是否靠增資增加。
- 是否靠重估資產增加。
- 保留盈餘是否穩定累積。
- 每股淨值是否增加。

## 一起看更完整

| 狀況 | 可能解讀 |
| --- | --- |
| 自由現金流為正、股東權益增加 | 體質可能穩定改善 |
| 自由現金流為負、股東權益增加 | 可能在投資成長，也可能收款壓力 |
| 自由現金流為負、股東權益下降 | 風險升高 |
| 高配息但自由現金流不足 | 配息可持續性要查證 |

## 記憶方法

自由現金流像真正能自由使用的零用錢。

股東權益像公司長期累積下來的家底。

## 本節小結

- 自由現金流能檢查公司是否真的產生可運用現金。
- 股東權益能觀察長期累積價值。
- 兩者一起看，比只看淨利更能判斷體質。
