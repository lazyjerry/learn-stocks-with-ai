# 第五堂：財務指標與公司體質 | 第 2 節：ROE、ROA、負債比

## 本節目標

- 用 ROE 看股東資金報酬。
- 用 ROA 看資產使用效率。
- 用負債比看財務槓桿與承壓能力。

重要提醒：

- 高 ROE 要拆開看來源。
- 高報酬可能來自好生意，也可能來自高槓桿。

## ROE：股東資金報酬

```text
ROE = 稅後淨利 / 股東權益
```

ROE 代表公司使用股東資金創造報酬的效率。

觀察重點：

- 是否長期穩定。
- 是否高於同業。
- 是否靠負債放大。

## ROA：全部資產使用效率

```text
ROA = 稅後淨利 / 資產總額
```

ROA 看的是公司使用全部資產創造獲利的能力。

如果 ROE 高但 ROA 普通，可能代表公司使用較多負債。

## 負債比：財務壓力

```text
負債比 = 負債總額 / 資產總額
```

負債比高，代表公司資產中有較高比例來自外部資金。

可能優點：

- 放大成長速度。
- 提高股東資金報酬。

可能風險：

- 利息負擔增加。
- 景氣反轉時壓力變大。
- 借款到期需要再融資。

## 三個指標一起看

| 狀況 | 可能解讀 |
| --- | --- |
| ROE 高、ROA 高、負債合理 | 體質可能較佳 |
| ROE 高、ROA 低、負債高 | 槓桿放大報酬 |
| ROE 低、ROA 低 | 資產使用效率可能差 |
| 負債比快速上升 | 要查擴張或壓力來源 |


## 記憶方法

ROE 像股東資金報酬率。

ROA 像公司全部資產的使用效率。

負債比像背包重量：

```text
背包重可以走更遠，也可能走到一半先累倒。
```

## 自我檢查

- ROE 高為什麼不一定代表安全？
- ROA 和 ROE 的差異是什麼？
- 負債比高可能有什麼優點與風險？
- 為什麼要和同業比較？

## 本節小結

- ROE 看股東資金報酬。
- ROA 看全部資產效率。
- 負債比協助判斷 ROE 是否靠槓桿放大。
