# 第五堂：財務指標與公司體質 | 第 1 節：營收、毛利、營業利益

> 免責聲明：本課程內容就是 AI 生成拿來學習用的內容，不構成任何投資建議、買賣建議或報酬保證。投資前請自行查證資料並評估風險。

## 本節目標

- 用營收看公司規模與需求。
- 用毛利看產品競爭力。
- 用營業利益看本業效率。

重要提醒：

- 營收創高但獲利沒跟上，要追原因。
- 公司體質不是看單一數字，而是看趨勢與結構。

## 營收看規模

營收代表公司賣出產品或服務取得的收入。

觀察方式：

- 年增率：跟去年同期比。
- 月增率或季增率：跟上一期比。
- 長期趨勢：是否穩定成長。

要追問：

- 成長來自價格、銷量，還是匯率？
- 是否只靠單一客戶？
- 成長是否可持續？

## 毛利看競爭力

毛利是營收扣掉直接成本。

```text
毛利率 = 毛利 / 營收
```

毛利率高，通常代表公司有較好的產品力、定價能力或成本控制。

毛利率下滑可能代表：

- 競爭變激烈。
- 成本上升。
- 產品組合變差。
- 客戶砍價。

> **Claude 按：** 毛利率跨產業差異極大：IC 設計業可達 50-60%+，LCD 面板代工可能低於 15%，連鎖超市約 20-30%。**務必只在同產業內比較**，或與同公司歷史區間比。毛利率持續高於同業且維持穩定，才是真正競爭護城河的訊號；若毛利率開始趨勢性下滑，通常代表競爭格局或定價能力已悄悄發生變化。

## 營業利益看本業效率

營業利益是扣掉營業費用後的本業成果。

```text
營益率 = 營業利益 / 營收
```

營益率改善可能代表：

- 規模放大後費用率下降。
- 產品組合改善。
- 管理效率提升。

營益率惡化可能代表：

- 費用成長太快。
- 研發或行銷投入增加。
- 毛利率下滑。

## 三個指標一起看

| 狀況 | 可能解讀 |
| --- | --- |
| 營收成長、毛利率上升、營益率上升 | 體質可能改善 |
| 營收成長、毛利率下降 | 可能用低價換成長 |
| 營收成長、營業利益不成長 | 費用或成本吃掉成果 |
| 營收下降、營益率改善 | 可能主動放棄低利產品 |

重要提醒：

- 指標變化要搭配公司說明與產業狀況。

- 為什麼營業利益比業外收益更重要？

- 分析公司體質時為什麼要看多年趨勢？

## 本節小結

- 營收、毛利、營業利益是理解本業體質的核心。
- 營收看規模，毛利看競爭力，營業利益看效率。
- 三者要一起看，不能只看單一數字。

> **Claude 按：** 台股**月營收公告制度**讓分析師可在季報前掌握趨勢。標準分析節奏是：月營收確認方向 → 季報確認毛利率 → 年報確認全年結構。三個時間層次的資料配合，是台股基本面分析的典型工作流程。「月營收年增率連續 3 個月轉正」常被作為基本面可能改善的初步觀察訊號，但仍需季報毛利率確認。
